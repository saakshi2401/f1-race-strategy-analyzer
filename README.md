# 🏎️ Formula 1 Race Strategy Analyzer

## 📌 Overview
This project analyzes Formula 1 race strategies using real-world lap data. It focuses on tire strategies, lap performance, and position changes.

## ⚙️ Tech Stack
- Python (Pandas, FastF1)
- Power BI

## 📊 Features
- Tire strategy visualization
- Lap time analysis
- Position change tracking
- Undercut strategy detection (based on tire changes)
